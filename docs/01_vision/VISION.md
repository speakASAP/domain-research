# Vision

Build an Alfares-native domain research service that turns a user's business description into domain candidates, checks availability and registration expiry evidence, schedules future rechecks, and notifies the user when a watched domain becomes available.

The service must integrate with the Alfares ecosystem instead of duplicating platform services: Auth for identity, ai-microservice for LLM calls, notifications-microservice for delivery, logging-microservice for logs, docs-rag-microservice for documentation retrieval, Vault for secrets, and Kubernetes for deployment.
